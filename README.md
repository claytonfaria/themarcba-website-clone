# claytonfaria.dev

My portfolio built with Next.js, Chakra UI, and Vercel.

## Running Locally

```bash
$ git clone https://github.com/claytonfaria/claytonfaria.dev.git
$ cd claytonfaria.dev
$ npm install
$ npm run dev
```

## Built Using

- [Next.js](https://nextjs.org/)
- [Vercel](https://vercel.com)
- [Chakra UI](https://chakra-ui.com/)
